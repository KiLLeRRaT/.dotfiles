# i3

## Config Location
`/i3-manjaro`

## Key Bindings

Typically, i3 keys are `Alt + Ctrl +  ...`

`Mod` is set to `Alt`

| Key              | Action        |
|------------------|---------------|
| `Mod`            | Alt           |
| `Mod + Ctrl + h` | Move to left  |
| `Mod + Ctrl + j` | Move to below |
| `Mod + Ctrl + k` | Move to above |
| `Mod + Ctrl + l` | Move to right |
